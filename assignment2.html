Assignment 2 One activity I did was all about web browsers. All web browsers are used to let users view and explore information across the web. Web browsers are used in lots of people’s everyday lives. I learned that web browsers are built up of many components but the main 3 that make it all function properly are called the Controller/Dispatcher, Interpreter, and the Client Programs. The controller takes the input from the mouse and keyboard and displays it onto the screen, which is how you can move a cursor and type words. An interpreter uses optional JAVA commands to suggest things that the user may be trying to do. Lastly, the client program gives command of the system which allows it to do a specific service. Whether it be building a website with picture pop up commands or something else like that, Client programs are very important for computers and are very common in lots of things on the internet. I apply these concepts to my environment and everyday because I use web browsers for all types of things on both my phone and computer every single day whether I’m looking something up or trying to get onto a certain website.
